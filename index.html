Veille technologique

Évolution des agents IA

Contexte IA & LLM

Qu'est-ce qu'un agent IA ?

Un agent IA est un système autonome alimenté par un LLM capable de planifier, mémoriser et utiliser des outils pour atteindre un objectif — sans intervention humaine constante. Contrairement à un chatbot qui répond, l'agent agit : il peut ouvrir un navigateur, consulter une source, synthétiser les données et envoyer une alerte de lui-même.

Source →
1950 – 1990 Histoire de l'IA

G1 — L'IA symbolique et les systèmes experts

Les premiers agents reposaient sur des règles "si/alors" codées manuellement, sans aucune capacité d'apprentissage. Le programme MYCIN (1972) diagnostiquait des maladies infectieuses via 600 règles médicales figées — hors de ce domaine, le système était inutile.

Source →
1990 – 2020 Machine Learning

G2 — L'apprentissage par renforcement

Les agents apprennent par essai/erreur en maximisant une récompense dans un environnement simulé. AlphaGo (DeepMind, 2016) a battu le champion du monde de Go — mais restait incapable d'effectuer la moindre autre tâche. Performant, mais enfermé dans un domaine unique.

Source →
Oct. 2022 LLM & Agents

G3 — ReAct : raisonnement + action en alternance

Le paradigme ReAct (Google, 2022) permet à un LLM de générer en alternance des traces de raisonnement verbal et des actions concrètes. L'agent cherche une info, raisonne sur ce qu'il trouve, corrige sa trajectoire, puis répond — réduisant drastiquement les hallucinations par rapport à un LLM classique.

Source →
2022 – 2025 Frameworks

L'écosystème des frameworks agents

LangChain (2022) a ouvert la voie à l'orchestration LLM. AutoGen (Microsoft, 2023) a introduit la collaboration multi-agents. CrewAI (2024) y a ajouté des rôles métier définis. LangGraph (2024) structure les workflows en graphes d'états cycliques. L'OpenAI Agents SDK (mars 2025) simplifie l'ensemble avec un framework natif et léger.

Source →
Nov. 2024 Standardisation

MCP — Le protocole universel agent-outil

Le Model Context Protocol (Anthropic, nov. 2024) est le "port USB-C de l'IA" : un standard ouvert qui connecte n'importe quel agent à n'importe quelle source de données sans intégration sur mesure. En mars 2025, OpenAI l'adopte. En avril, Google DeepMind suit. En décembre, le standard est cédé à la Linux Foundation.

Source →
2025 – 2026 Tendances

Agents long horizon et consolidation du marché

Le marché converge vers des agents capables de travailler en autonomie plusieurs heures : Devin code une application entière sans supervision. Côté frameworks, Microsoft fusionne AutoGen et Semantic Kernel. Le marché des agents atteint 7,84 Md$ en 2025, avec une projection à 52,62 Md$ d'ici 2030.

Source →
2024 – 2026 Sécurité

Hallucination, coût et sécurité des agents

Les agents restent vulnérables aux hallucinations et aux attaques de "tool poisoning" — des instructions malveillantes cachées dans un outil peuvent forcer l'agent à exfiltrer des données sensibles. À cela s'ajoute un coût token élevé qui freine l'adoption en entreprise. La supervision humaine reste indispensable.
